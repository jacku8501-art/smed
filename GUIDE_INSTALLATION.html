🏭 Guide d'installation — SMED MDC Confection

Configure Firebase et installe l'app sur ton téléphone Android en 10 minutes

① Créer un compte Firebase (gratuit)

Va sur console.firebase.google.com et connecte-toi avec ton compte Google.

Clique sur "Créer un projet"

  • Nom du projet : smed-mdc-confection
  • Google Analytics : tu peux désactiver
  • Clique "Créer le projet"

② Ajouter une application Web

Dans ton projet Firebase :

  • Clique sur l'icône </> (Web) pour ajouter une app web
  • Nom : SMED MDC
  • Coche "Firebase Hosting" si tu veux héberger en ligne
  • Clique "Enregistrer l'application"

👉 Copie la configuration qui apparaît — elle ressemble à ça :

{
  "apiKey": "AIzaSy...",
  "authDomain": "smed-mdc-confection.firebaseapp.com",
  "databaseURL": "https://smed-mdc-confection-default-rtdb.firebaseio.com",
  "projectId": "smed-mdc-confection",
  "storageBucket": "smed-mdc-confection.appspot.com",
  "messagingSenderId": "123456789012",
  "appId": "1:123456:web:abcdef"
}

③ Activer la Realtime Database

Dans le menu gauche de Firebase :

  • Clique sur "Realtime Database"
  • Clique "Créer une base de données"
  • Choisis la région : Belgium (europe-west1)
  • Mode : Commencer en mode test

Ensuite va dans l'onglet "Règles" et mets :

{
  "rules": {
    ".read": true,
    ".write": true
  }
}

Clique "Publier"

⚠️ Ces règles permettent à tous d'accéder aux données. Pour un usage professionnel, tu pourras ajouter une authentification plus tard.

④ Lancer l'application

Double-clique sur index.html dans le dossier smed_app.

L'écran de configuration apparaît — colle ta config Firebase et clique "Connecter Firebase".

✅ Si tu vois "● En ligne" en haut à droite, Firebase est connecté !

⑤ Héberger en ligne (pour accès depuis le téléphone)

Pour accéder à l'app depuis ton téléphone, tu as besoin d'une URL publique. La solution la plus simple :

Option A — GitHub Pages (gratuit, recommandé) :

  • Crée un compte sur github.com
  • Crée un nouveau dépôt public
  • Upload les fichiers du dossier smed_app
  • Va dans Settings → Pages → Source : main branch
  • Ton URL sera : https://ton-nom.github.io/smed-mdc

Option B — Firebase Hosting (gratuit) :

  • Installe Firebase CLI : npm install -g firebase-tools
  • Dans le dossier smed_app : firebase deploy

⑥ Installer sur Android (depuis Chrome)

  • Ouvre l'URL de l'app dans Chrome sur ton téléphone
  • Appuie sur le menu ⋮ (3 points)
  • Sélectionne "Ajouter à l'écran d'accueil"
  • L'app s'installe comme une vraie application !
✅ L'app apparaît sur ton écran d'accueil avec l'icône MDC. Elle synchronise les données en temps réel avec le PC !

⑦ Générer un vrai fichier .APK (optionnel)

Si tu veux un vrai fichier APK installable :

  • Va sur pwabuilder.com
  • Colle l'URL de ton app hébergée
  • Clique "Package for stores" → Android
  • Télécharge le fichier .apk
  • Installe-le sur ton téléphone (autoriser les sources inconnues)
💡 Le fichier APK généré par PWABuilder est une vraie app Android basée sur WebView — identique à l'app web mais packagée comme une vraie app.
